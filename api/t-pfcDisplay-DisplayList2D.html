Class pfcDisplayList2D


Description

The two-dimensional Display List

To make your 2D graphics permanent create this object and place your graphics calls in its listener.

Forces the display of the DisplayList2D using the specified transformation matrix

Exceptions thrown (but not limited to):

pfcXToolkitNotExist - The display list is empty.

pfcXToolkitNotFound - The function could not find a display list with the specified identifier.

pfcXToolkitNotValid - The specified display list is invalid.
