Class pfcRelationFunctionOptions


Description

Options to enable/disable various relation function related features.


Property Summary

/* optional */ pfcRelationFunctionArgumentsArgumentTypes
The types of arguments permitted by the relation function. Default is null, indicating that no arguments are permitted.
/* optional */ booleanEnableArgumentCheckMethod
Turn the method (using pfcRelationFunctionListener.CheckArguments()) on/off. Default is false.
/* optional */ booleanEnableExpressionEvaluationMethod
Turn the method (using pfcRelationFunctionListener.EvaluateFunction()) on/off. Default is true.
/* optional */ booleanEnableTypeChecking
Turn argument type checking (using pfcRelationFunctionOptions.ArgumentTypes) on/off. Default is false.
/* optional */ booleanEnableValueAssignmentMethod
Turn pfcRelationFunctionListener.AssignValue() on/off. Default is false.
